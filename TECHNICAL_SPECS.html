Infraestructura & Backend

  • Supabase (BaaS)

    Utilizado como motor principal de datos (PostgreSQL 15), sistema de autenticación JWT y almacenamiento de archivos planos (S3-compatible bucket).

  • Realtime Engine

    Suscripciones a bases de datos vía WebSockets para actualización instantánea de métricas y estados de propiedades en el dashboard.

  • Edge Computing

    Despliegue global optimizado para latencia mínima en carga de activos y ejecución de lógica de frontend.

  • Computer Vision: Extracción paramétrica de metros cuadrados, número de habitaciones y acabados arquitectónicos directamente de fotografías.

    • Matriz de Funcionalidades

  • Módulo 360° (Pannellum SDK): Visor panorámico inmersivo integrado con aceleración por hardware (WebGL) para tours virtuales sin latencia.
  • Engine de Reportes PDF: Integración de jsPDF + html2canvas para generación de fichas técnicas ("Brochures") en tiempo de ejecución del lado del cliente.
  • GIS & Mapping: Implementación de Leaflet.js con capas de OpenStreetMap para georreferenciación precisa y análisis de entorno.
  • CRM Inteligente: Sistema de gestión de leads con persistencia en PostgreSQL y triggers automatizados para seguimiento vía WhatsApp API Business.

    • Seguridad & Protocolos

  • Encriptación: Datos en reposo encriptados mediante AES-256 (Supabase Storage).
  • Control de Acceso: Row Level Security (RLS) implementado en base de datos para garantizar el aislamiento estricto de datos entre diferentes agencias.
  • Auth Flow: Autenticación basada en tokens JWT con tiempos de expiración configurables para máxima seguridad.
